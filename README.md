# CodeAlpha Task 1 - Hangman Game

A simple text-based Hangman game built in Python.

## Features

- Uses a predefined list of 5 words
- Allows 6 incorrect guesses
- Accepts one-letter console input
- Tracks guessed letters
- Shows win or game-over result

## How to Run

```bash
python hangman.py
```

## Concepts Used

- `random`
- `while` loop
- `if-else`
- strings
- lists
